Exit SEIS record

Exit an active SEIS record either because the student is no longer a SWD, or the student is still a SWD but has left the district.

Select one:

  • Exit 70 No Longer Eligible
  • Exit 78 Written Revocation
  • Exit 76/87 Transfer out
  • Exit 77 Deceased


* DO NOT USE Exit Reason 74 - Drop Out

* DO NOT USE Exit Reason 78 if the parent has not provided written revocation of services. Written revocation must include student first and last name, student DOB, parent's name, parent's signature, date, and a state that the parent is withdrawing the student from SpEd services.

* "EXITED SPED OUT OF STATE" is only to be used if the student was a SWD in California, moved out of CA and was exited SpEd outside of CA, then returned to CA. Upon return, CALPADS is not aware that the student was exited from SpEd, so this exit reason is used to report that the student is no longer a SWD.
